# AcadSync

AcadSync is a role-based College Management System frontend built using **React + Vite**.

## Features
- Role-based dashboards:
  - Admin
  - Student
  - Faculty
- Modern UI with smooth navigation
- Fast development using Vite
- Scalable React project structure

## Tech Stack
- React
- Vite
- JavaScript
